1. 시스템 요구사항
================================================================================

  - 운영체제: Windows 10 (64비트) 이상
  - 인터넷 연결: 필수 (거래소 데이터 조회용)
  - 디스크 공간: 약 200MB
  - 추가 설치: 없음 (.NET 런타임 포함됨)


================================================================================
2. 설치 방법
================================================================================

  [방법 1] ZIP 파일로 받은 경우

  1) 받은 ZIP 파일을 원하는 위치에 압축 해제합니다.
     예: C:\Games\RoMarketCrawler\

  2) 압축 해제 후 다음 파일들이 있는지 확인합니다:
     - RoMarketCrawler.exe (메인 프로그램)
     - Data 폴더 (데이터 파일)

  3) 설치 완료!


  [방법 2] 폴더로 받은 경우

  1) 받은 폴더를 원하는 위치에 복사합니다.
  2) 설치 완료!


================================================================================
3. 프로그램 실행
================================================================================

  1) RoMarketCrawler.exe를 더블클릭하여 실행합니다.

  2) 처음 실행 시:
     - 네트워크 연결을 확인합니다.
     - 프로그램 사용 동의를 요청합니다.
     - 아이템 인덱스를 자동으로 생성합니다 (약 1-2분 소요).

  3) 이후 실행 시:
     - 바로 메인 화면이 표시됩니다.


================================================================================
4. 주의사항
================================================================================

  [보안 경고]

  - Windows SmartScreen 경고가 나타날 수 있습니다.
  - 이는 프로그램이 서명되지 않아서 나타나는 정상적인 경고입니다.
  - "추가 정보" 클릭 후 "실행" 버튼을 누르시면 됩니다.


  [방화벽 설정]

  - 처음 실행 시 Windows 방화벽 허용 요청이 나타날 수 있습니다.
  - "허용"을 선택해주세요 (인터넷 연결에 필요합니다).


  [중복 실행 방지]

  - 프로그램은 한 번에 하나만 실행할 수 있습니다.
  - 이미 실행 중인 경우 "중복 실행" 메시지가 나타납니다.


  [데이터 저장 위치]

  - 설정 및 관심 아이템 데이터는 프로그램 폴더에 저장됩니다:
    - settings.json: 프로그램 설정
    - watch_list.json: 관심 아이템 목록
    - price_history.db: 가격 기록

5. 문제 해결
================================================================================

  Q: 프로그램이 실행되지 않아요.
  A: Windows SmartScreen에서 차단되었을 수 있습니다.
     "추가 정보" -> "실행"을 클릭하세요.

  Q: "네트워크 연결이 없습니다" 오류가 나타나요.
  A: 인터넷 연결을 확인하세요.
     방화벽에서 프로그램이 차단되었는지 확인하세요.

  Q: "사용 기간이 만료되었습니다" 오류가 나타나요.
  A: 새 버전을 요청해주세요.

  Q: 아이템 검색이 느려요.
  A: 처음 실행 시 인덱스 생성이 필요합니다.
     잠시 기다려주세요.

  Q: 설정을 초기화하고 싶어요.
  A: 프로그램 폴더에서 settings.json 파일을 삭제하세요.
